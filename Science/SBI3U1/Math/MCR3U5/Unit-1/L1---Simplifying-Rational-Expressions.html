    Nov 22, 20243 min read

    Notes

    What is a polynomial?

    A polynomial is an algebraic expression formed from addition, substraction, or multiplication of terms

    Example

    - Binomial
    - Trinomial
    - Monomial

    What is a rational expression?

    A rational expression is an algebraic expression that is expressed under the form in which and are polynomials (terms with coefficients, variables, and exponents) and

    How to simplify the rational expression

    1. Factorize the expression

    In order to simplify a rational expression, we must factorize both the numerator and denominator.

    2. State the restrictions

    Then, we must state the restrictions in the denominator.

    What is a restriction?

    A restriction is a value that renders the expression undefined.

    Hence, a value that renders the denominator equal to zero.

    3. Simplify the expression

    Much like fractions, we can then simplify rational expressions by utilizing common factors.

    Example

    Discontinuities

    There are 2 types of discontinuities in a graph that can correspond to a restriction

    Open Point

    Point on graph that is undefined (shown by a hollow circle).

    An open point corresponds to a common factor that is simplified from the numerator and denominator

    Vertical Asymptote

    A vertical line that the function never crosses.

    A vertical asymptote corresponds to a factor that is uniquely in the denominator after everything is simplified
