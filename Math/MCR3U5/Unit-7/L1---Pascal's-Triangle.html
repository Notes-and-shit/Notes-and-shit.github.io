    L1 - Pascal's Triangle

    Jun 17, 20243 min read

    Notes

    What is Pascal's Triangle?

    Pascal’s Triangle is a triangle formed by adding adjacent numbers from the previous rows to form the numbers below them.

    Properties:

    • The number of rows (n) starts at n=0
    • The first and last term of each row is 1
    • Each term is the sum of the two adjacent numbers of the previous row

    Key Formulas

    Number of Rows (n)=n
    Number of Terms=n+1
    Sum of Terms=2n

    Notation:

    we start at 0

    n=rows
    r=horizontalposition

    3 ways to represent a term in a triangle:

    1. tn,r​
    2. n​Cr​
    3. (rn​)

    Pascal’s Formula:

    Each term is the sum of the 2 adjacent pairs from the previous row:

    1. tn,r​ = tn−1,r−1​ + tn−1,r​
    2. n​Cr​=n−1​Cr−1​+n−1​Cr​
    3. (rn​) = (r−1n−1​) + (rn−1​)

    Evaluate using factorials:

    The factorial of n is the product of n and all the whole values smaller than it.

    n!=n(n−1)(n−2)...3∗2∗1

    tn,r​ = n​Cr​ = (rn​) = r!(n−r)!n!​
