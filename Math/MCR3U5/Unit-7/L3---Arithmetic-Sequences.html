    What is a sequence?

    A sequence is a list of numbers, figures, letters, etc. that are organized in a distinct and identifiable order

    Each number in a sequence is called a term. A sequence can be infinite or finite.

    What is an arithmetic sequence

    An arithmetic sequence is a sequence where the difference in between 2 consecutive terms is constant

    This means that we can always add/subtract the same value to find the next term
